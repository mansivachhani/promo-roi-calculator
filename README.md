# iGaming PO Starter

A lean Next.js + Tailwind starter for product demos, calculators, and analytics dashboards.

## Quick Start

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Deploy (Vercel)

```bash
npx vercel
```

## Suggested Add-ons

- Charts: Recharts or Chart.js
- Data: local JSON or Supabase free tier
- CI: GitHub Actions (lint + build)
